> 每个官职为一个槽位：有官员时显示该官员的角色卡（立绘+姓名），空缺时显示官职名
>
> 点击有官员的槽位→弹出角色详情(3.4)；点击空缺槽位→弹出填补界面
>
> 中枢线以上（正三品及以上）为中枢官员，其角色卡带稳固值条
>
> 本族官员的角色卡边框高亮标识
>
> 路连线到下辖各州，州连线到下辖各县，连线横平竖直

**功能要素：**

| 要素 | 说明 |
|------|------|
| **官职金字塔** | 上方中枢按品阶从上到下排列（正一品→正三品），下方地方按路→州→县层级排列 |
| **角色卡槽位** | 有官员显示角色卡，空缺显示官职名；点击有官员→角色详情(3.4)，点击空缺→填补界面 |
| **中枢线** | 正三品下方分隔，中枢官员（正三品及以上）角色卡带稳固值条 |
| **路州县连线** | 路→州→县逐级连线，横平竖直，同一路的官员聚合在一起 |
| **本族标识** | 本族官员角色卡边框高亮 |
| **科举倒计时** | 右上角显示下次科举时间 |

**系统关联：**

```
官职总览
  ├→ 官场系统：品阶链、空缺填补、政绩槽
  ├→ 角色系统：官员角色卡展示
  ├→ 时间系统：科举倒计时、述职倒计时
  └→ 党争系统（中枢后）：稳固值显示
```

**攻击操作流程：**

```

  ├→ 攻讦/检举/离间/弹劾各有独立弹窗
```

**地图底图：**

| 要素 | 设计 |

| **风格** | 水墨山水画风格，宣纸质感底色，山川河流以淡墨勾勒 |
| **地域轮廓** | 简化的中国中部偏东区域轮廓，突出京畿/江南/岭南三路，其余区域淡化留白 |
| **地形元素** | 山脉（淡墨晕染）、河流（浅蓝细线）、海岸线（淡蓝渐变） |
| **季节变化** | 底图随季节微变：春·淡绿柳烟、夏·浓荫碧水、秋·枯黄落叶、冬·雪景留白——仅色调微调，不影响地图功能 |

| **拖拽** | 缩放超过100%后可拖拽移动地图：按住上下左右方向键拖动，或将鼠标移至屏幕边缘自动向该方向滚动 |
