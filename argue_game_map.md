## 一、行政区划总览

```
京畿路                江南路               岭南路
 ├─ 开封府(州)         ├─ 江宁府(州)        ├─ 广州府(州)
 │   ├─ 祥符县         │   ├─ 上元县         │   ├─ 南海县
 │   └─ 尉氏县         │   └─ 江宁县         │   └─ 番禺县
 ├─ 河南府(州)         └─ 苏州(州)          └─ 韶州(州)
 │   ├─ 河南县             ├─ 吴县              ├─ 曲江县
 │   └─ 洛阳县             └─ 长洲县            └─ 乐昌县
  └─ 应天府(州)
      ├─ 应元县
      └─ 溧阳县
```

- 3路 × 3/2/2州 × 2县 = 共7州14县
- 地方官总数：14县×3职 + 7州×4职 + 3路×4职 = 42+28+12 = 82人

## 五、县域特殊修正

部分县域因地理位置或特殊条件，拥有额外修正：

| 县 | 修正类型 | 效果 | 说明 |
|----|---------|------|------|
| 溧阳县 | 政绩槽修正 | -5% | 地势偏僻，治理难度较大 |
| 乐昌县 | 政绩槽修正 | -5% | 边境小县，资源匮乏 |
| 江宁县 | 金钱建筑产出 | +5% | 水运便利，商贸更旺 |
| 长洲县 | 金钱建筑产出 | +5% | 丝织重镇，商品附加值高 |
| 吴县 | 官学成长 | +5% | 文教名县，求学氛围浓厚 |
| 番禺县 | 武馆成长 | +5% | 军事要冲，习武风气盛 |
| 乐昌县 | 武馆成长 | +5% | 边境驻防，军事训练强化 |

- 特殊修正与繁荣度修正叠加计算
- 繁荣度变化后特殊修正不改变（由地理/文化条件决定，永久生效）

## 七、路与党派的关系

- 畿党与淮党在不同路的势力基础不同，影响NPC家族分布：
  - 畿党根基：京畿路（靠近中枢，政治资源强）
  - 淮党根基：江南路（经济富庶，金钱资源强）
  - 岭南路：两党势力均弱，为争夺区域

## 八、地图与系统的关联

| 系统 | 关联方式 |
|------|---------|
| **官场系统** | 县/州/路三级官职与地图对应；空缺填补受地理位置约束 |
| **建筑系统** | 州城城市建筑配置见第三节；繁荣度影响建筑等级和产出 |
| **税收系统** | 繁荣度决定税额；路级官员从下辖州县抽成 |
| **党争系统** | 畿党与淮党在不同路的势力基础不同；岭南路为争夺区 |
| **事件系统** | 边境县触发军事事件；沿海城市触发海贸事件；京城触发党争事件 |
| **剧情系统** | 主角灭门地点：开封府沈家；科举地点：开封府；复仇地点：中枢（京城） |
