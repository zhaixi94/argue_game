# 官场党争复仇游戏 — AI家族行为逻辑设计文档

---

## 一、设计原则

1. **每个NPC家族独立决策**——根据自身资源、官职、党派、关系状态做出合理选择
2. **决策有优先级**——生存>发展>党争，资源不足时自动降级目标
3. **决策有个性差异**——不同家族有不同的战略倾向（重经济/重政治/重军事），不按同一套模板行动
4. **决策可被玩家感知**——NPC行动有迹可循，玩家能通过观察推断NPC意图
5. **决策不完美**——NPC不会做最优解，有随机波动和失误空间，避免"完美AI"体验

---

## 二、NPC家族档案与战略倾向

每个NPC家族有固定的战略倾向，影响其决策权重。

### 2.1 战略倾向类型

| 倾向 | 侧重 | 建筑/行动偏好 | 代表家族 |
|------|------|-------------|---------|
| **政治型** | 官职升迁、声望积累、党争攻防 | 优先建声望类/圣眷类建筑，积极参政 | 陆家、周家、苏家 |
| **经济型** | 金钱积累、商业扩张、资源碾压 | 优先建金钱类建筑，大量入驻城市商铺 | 钱家、林家、赵家 |
| **军事型** | 军事成长、武职升迁、边疆经营 | 优先建演武场，争抢武馆/武职 | 方家、何家、陈家 |
| **中立型** | 均衡发展、观望局势 | 无明显偏好，随机波动 | 顾家、梁家 |

| 家族 | 党派 | 所在 | 声望 | 倾向 | 性格标签 | 行为特征 |
|------|------|------|------|------|---------|---------|

## 三、AI决策总框架

### 3.1 每回合决策流程

```
月初：评估阶段
  ├─ 计算本回合可用资源点、金钱、声望、圣眷
  ├─ 评估家族当前状态（危局/正常/优势）
  └─ 确定本回合优先目标
       ↓
决策阶段：按优先级执行
  优先级1: 生存维护（资源点分配、防御预设）
  优先级2: 发展经营（建筑建造/升级、角色放置、城市入驻）
  优先级3: 家族行动（联姻、结交、商议）
  优先级4: 党争行动（攻防、策反、情报）
       ↓
月末：检查阶段
  ├─ 检查是否有遗漏的必做操作
  └─ 未用完的资源点投入产出最高的建筑
```

### 3.2 家族状态评估

| 状态 | 条件 | 行为调整 |
|------|------|---------|
| **危局** | 金钱<50贯 或 戒心≥60 或 有官员稳固值<30 | 暂停所有非必要支出，全力防御/降戒心 |
| **紧张** | 金钱<200贯 或 戒心≥30 或 政绩槽<60% | 减少进攻性行动，优先资源积累 |
| **正常** | 不满足危局/紧张条件 | 按战略倾向正常行动 |
| **优势** | 金钱>1000贯 且 声望>5000 且 无官员受威胁 | 可主动发起攻势、扩张联姻 |

## 四、资源点分配逻辑

### 4.1 分配优先级

NPC家族按以下优先级分配每月资源点：

| 优先级 | 用途 | 分配量 | 条件 |
|--------|------|--------|------|
| **1** | 官职防御预设 | 中枢官员每人1~3点 | 有中枢官员时 |
| **2** | 政绩槽加成（资源点投入） | 地方官每人1~2点 | 有地方官且政绩槽<100%时 |
| **3** | 金钱类建筑投入 | 优先高产建筑2~3点 | 需要金钱时 |
| **4** | 声望/圣眷类建筑投入 | 1~2点 | 需要声望/圣眷时 |
| **5** | 成长类建筑投入 | 1点 | 有空余资源点时 |
| **6** | 党争行动消耗 | 1~3点 | 优势状态且有目标时 |

### 4.2 分配策略差异

| 倾向 | 金钱建筑 | 声望建筑 | 圣眷建筑 | 成长建筑 |
|------|---------|---------|---------|---------|
| 政治型 | 1~2点 | 2~3点 | 2~3点 | 1点 |
| 经济型 | 2~3点 | 1~2点 | 1点 | 1点 |
| 军事型 | 1点 | 1点 | 1~2点 | 演武场2~3点 |
| 中立型 | 1~2点 | 1~2点 | 1~2点 | 1点 |

## 五、建筑决策逻辑

### 5.1 家族建筑建造

**触发条件：** 家族建筑数未达上限 且 金钱>建造费用×2（留有余量）

**建造优先级：**

| 倾向 | 第1栋 | 第2栋 | 第3栋 | 第4栋+ |
|------|-------|-------|-------|--------|
| 政治型 | 书院 | 书院 | 私塾 | 祠堂→论辩台→御前馆 |
| 经济型 | 商号 | 私塾 | 书院 | 祠堂→梳妆阁→钱庄 |
| 军事型 | 演武场 | 私塾 | 书院 | 论辩台→祠堂→商号 |
| 中立型 | 私塾 | 书院 | 演武场 | 按需随机 |

- 中枢后解锁圣眷类/情报类建筑，所有倾向优先补建御前馆+暗阁

### 5.2 家族建筑升级

**触发条件：** 金钱>升级费用×1.5 且 声望>升级声望消耗×1.2

**升级优先级：**

1. 当前产出最高的建筑优先升级
2. 多栋同级时，按倾向偏好排序
3. 不升级利用率低的建筑（无角色放置的建筑）

### 5.3 城市建筑入驻

**入驻逻辑：**

```
1. 筛选可用槽位：该州城有空槽 且 家族入驻未达上限
2. 选择建筑类型：按战略倾向权重随机选择
   - 经济型：金钱类60% > 声望类25% > 成长类15%
   - 政治型：声望类40% > 金钱类30% > 成长类30%
   - 军事型：成长类(武馆)50% > 金钱类30% > 声望类20%
3. 选择入驻角色：该建筑对应能力最高的空闲角色
4. 投入资源点：金钱类必投1点，其他按倾向1~2点
```

**入驻撤出条件：**

| 条件 | 处理 |
|------|------|
| 金钱不足以支付入驻消耗 | 自动撤出消耗最高的建筑 |
| 入驻角色被任命官职 | 自动撤出，空出角色 |
| 有更高产出建筑空出槽位 | 转移入驻（每回合最多1次） |

## 六、角色放置逻辑

### 6.1 角色放置优先级

```
1. 担任官职的角色 → 不在建筑中（官职优先）
2. 高能力成年角色 → 按能力匹配最高产出的建筑
3. 低能力/年轻角色 → 成长类建筑
4. 无合适建筑的空闲角色 → 留空（不影响产出）
```

### 6.2 角色与建筑匹配

| 角色最高能力 | 推荐建筑 | 说明 |
|-------------|---------|------|
| 学识最高 | 书院/官学 | 学识对应声望产出 |
| 商业最高 | 商号/商铺/钱庄/漕运司 | 商业对应金钱产出 |
| 口才最高 | 论辩台/名士馆/诗社/御前馆 | 口才对应声望/圣眷产出 |
| 军事最高 | 演武场/武馆 | 军事对应军事成长/产出 |
| 样貌最高 | 梳妆阁 | 样貌成长，为联姻储备 |

- 匹配时取角色最高能力对应的建筑，若该建筑无空槽则取次高能力匹配

## 七、联姻决策逻辑

### 7.1 发起联姻条件

| 条件 | 说明 |
|------|------|
| 家族内有适龄未婚角色 | 15岁(男)/16岁(女)以上 |
| 金钱≥50贯 | 最低联姻投入门槛 |
| 目标家族有适龄异性角色 | 对方家族有空缺 |
| 非危局状态 | 危局时暂停联姻 |

### 7.2 目标家族选择

**评分公式：**

```
联姻目标评分 = 关系加成 + 声望加成 + 党派加成 + 战略加成

关系加成 = 与目标家族关系值 × 0.5（上限30分）
声望加成 = min(30, 目标家族声望 / 200)（倾向高声望家族）
党派加成 = 同党派+20 / 无党派+10 / 敌对党派-20
战略加成 = 目标家族有官职时+15 / 有中枢官职时+30
```

- 选评分最高的家族发起联姻
- 政治型家族额外+50%党派加成权重
- 经济型家族额外+50%声望加成权重

### 7.3 联姻投入

```
投入资源点 = min(3, 可用资源点)  ← 通常全力投入
投入声望 = min(200, 可用声望×30%)  ← 保留部分声望
投入金钱 = min(500贯, 可用金钱×40%)  ← 保留过半金钱
```

- 危局时减少投入：资源点1点，金钱100贯
- 目标家族声望远高于己方时，投入增加50%

## 八、结交决策逻辑

### 8.1 发起结交条件

| 条件 | 说明 |
|------|------|
| 有空闲成年角色可派出 | 非官职、非建筑槽位中的角色 |
| 金钱≥20贯 | 最低结交门槛 |
| 目标人物关系<80 | 已很高则不浪费资源 |
| 有明确结交目标 | 见8.2 |

### 8.2 结交目标选择

**优先级：**

| 优先级 | 目标 | 条件 |
|--------|------|------|
| **1** | 同党派家族中关系<30的族长 | 防止忠诚度过低 |
| **2** | 无党派家族族长 | 拉拢入党派 |
| **3** | 敌对党派家族中关系>20的官员 | 策反前铺垫 |
| **4** | 城市建筑槽位被占家族的族长 | 为商议铺路 |

**派出角色选择：** 家族中口才最高的空闲角色

### 8.3 结交频率

- 每2回合最多结交1次（避免资源浪费）
- 政治型家族每1回合可结交1次
- 危局时暂停结交

## 九、商议决策逻辑

### 9.1 发起商议条件

| 条件 | 说明 |
|------|------|
| 目标州城有被占槽位且家族入驻未满 | 有扩张空间 |
| 与占位家族关系≥20 | 关系太差不宜商议 |
| 金钱≥100贯 | 商议最低投入 |

### 9.2 商议类型选择

| 类型 | 选择条件 | 建筑类型约束 |
|------|---------|-------------|
| **分成** | 关系≥50 且 只需短期使用 | 仅资源类和特色建筑（成长类不可分成） |
| **转租** | 关系≥30 且 需要3年使用权 | 仅资源类和特色建筑（成长类不可转租） |
| **出让** | 关系≥40 且 金钱充裕(>500贯) 且 长期需要 | 所有建筑类型均可 |

- 成长类建筑（官学/武馆/茶楼）只能选择出让

### 9.3 商议目标选择

- 优先选择己方有官职的州城
- 优先选择产出最高的建筑类型（按战略倾向）
- 经济型家族：优先商议金钱类建筑
- 政治型家族：优先商议声望类建筑

## 十、党争决策逻辑

### 10.0 NPC攻防通用规则

- NPC之间攻防使用与玩家完全相同的公式（攻讦/检举/离间/弹劾），不简化
- 每个NPC家族每回合最多发起1次稳固值攻击（与玩家限制一致）
- NPC防御预设按10.2节逻辑执行
- NPC间攻防结果按标准公式即时结算

### 10.1 攻击目标选择

**优先级：**

| 优先级 | 目标 | 条件 |
|--------|------|------|
| **1** | 威胁己方官员的敌方官员 | 己方官员稳固值<50时优先反击 |
| **2** | 敌方中枢最低稳固值官员 | 稳固值最低的最容易击倒 |
| **3** | 敌方中枢最高品级官员 | 打击敌方核心 |
| **4** | 己方空缺可填补的目标 | 击倒后有升迁机会 |

**攻击方式选择：**

| 情况 | 选择方式 |
|------|---------|
| 有检举情报 且 目标稳固值<50 | 检举（必中收割） |
| 有离间情报 且 目标圣眷>30 | 离间（削圣眷涨戒心） |
| 有弹劾情报 且 目标戒心≥60 | 弹劾（高戒心时弹劾后更容易贬官） |
| 无情报 | 攻讦（高频磨血） |
| 目标稳固值<20 | 攒情报等检举/弹劾（不浪费攻讦） |

### 10.2 防御预设

**资源点分配：**

| 官员稳固值 | 投入资源点 | 投入圣眷 |
|-----------|-----------|---------|
| ≥80 | 0~1点 | 0点 |
| 50~79 | 1~2点 | 5~10点 |
| 30~49 | 2~3点 | 10~20点 |
| <30 | 3点 | 20~30点 |

- 戒心≥60时，同稳固值档位额外+1资源点或+5圣眷（因受伤系数高，更重视防御）
- 军事型家族防御投入偏少（性格刚猛，省资源进攻）
- 政治型家族防御投入偏多（稳健保守）

### 10.3 情报领取

**领取优先级：**

| 优先级 | 情报类型 | 条件 |
|--------|---------|------|
| **1** | 弹劾情报 | 有弹劾目标且党派贡献足够 |
| **2** | 检举情报 | 有检举目标 |
| **3** | 离间情报 | 有高圣眷目标 |
| **4** | 不领取 | 贡献不足时保留贡献用于兑换资源 |

- 弹劾情报最贵(30贡献)，优先级最高——与弹劾的高伤害匹配
- 情报卡持有达上限时不再领取

### 10.4 拉拢策反

**发起条件：**

| 条件 | 说明 |
|------|------|
| 目标家族党派忠诚<30 | 低于策反阈值 |
| 金钱≥200贯 | 策反最低投入 |
| 非危局状态 | 危局时无力策反 |

**目标选择：** 敌方忠诚最低的家族优先

**投入：** 资源点1~2点 + 金钱200~500贯（忠诚越低投入越少）

### 10.5 党派机构投入

| 机构 | 投入条件 | 投入量 |
|------|---------|--------|
| **谍报** | 始终投入 | 1~3点党派资源点（有攻击目标时3点，无目标时1点） |
| **舆论** | 有明确攻防需求时 | 发布与当前战略匹配的buff |
| **私库** | 被动 | 按领袖设定抽成，不主动存入（除非金钱极度充裕>2000贯） |

**舆论buff选择：**

| 需求 | 选择buff |
|------|---------|
| 己方官员被围攻 | 清名（防御+10%） |
| 准备集中攻击 | 流言（敌方攻讦+10%） |
| 需要圣眷积累 | 颂德（圣眷+10%） |
| 敌方戒心已高 | 疑云（额外戒心+2） |
| 准备大朝会 | 风骨（连线血量+50） |
| 敌方有家族动摇 | 倾轧（忠诚-3） |

## 十一、官职决策逻辑

### 11.1 科举报名

**报名条件：** 家族内15岁以上未参加过科举的成员

**报名逻辑：**

| 条件 | 处理 |
|------|------|
| 学识≥60 | 报名（大概率通过省试） |
| 学识50~59 | 报名（可能过解试，省试看运气） |
| 学识<50 | 不报名（浪费报名费） |

- 政治型家族：学识≥45即报名（更重视科举，愿意冒险）
- 军事型家族：学识≥55才报名（不浪费资源）

### 11.2 空缺填补

**地方官空缺：**

- 家族内有符合条件者（科举通过+品级匹配）时，按优先级投入资源点
- 投入量 = 基础5点 × (1 + 竞争者数量 × 0.5)
- 政治型家族额外+3点投入（更重视官职）

**中枢官职暗标：**

```
投入资源点 = min(可用资源点, 5)
投入圣眷 = min(可用圣眷×40%, 30)
投入声望 = min(可用声望×20%, 500)

暗标总值 = 资源点×10 + 圣眷×1 + 声望×0.5
```

- 政治型家族：圣眷投入比例提升至60%
- 危局时不参与中枢暗标（资源留给防御）

### 11.3 政绩槽资源投入

| 政绩槽状态 | 资源点投入 | 圣眷投入 |
|-----------|-----------|---------|
| <60%（丁等风险） | 3点 | 3~5点 |
| 60%~79%（丙等） | 2点 | 1~2点 |
| 80%~99%（乙等） | 1点 | 0点 |
| ≥100%（甲等） | 0点 | 0点 |

- 距述职<3月且<80%时，额外加投2点资源点

## 十二、税收截留决策

### 12.1 截留比例设定

| 性格标签 | 默认截留比例 | 调整条件 |
|---------|-------------|---------|
| 稳健 | 10% | 戒心≥40时降至5% |
| 清高 | 5% | 始终低截留 |
| 冒险 | 30% | 戒心≥60时降至20% |
| 精明 | 20% | 有截留情报产出时降至10% |
| 灵活 | 15% | 根据对手谍报等级动态调整 |
| 保守 | 5% | 始终最低截留 |
| 刚猛 | 20% | 不因戒心降低 |

**动态调整规则：**

- 若对立党派谍报机构等级≥2（中级）：截留比例-10%
- 若家族金钱<100贯：截留比例+10%（急需用钱）
- 截留比例范围：0%~50%

## 十四、大朝会自动结算

NPC参与大朝会时不进入小游戏，使用主文档中的自动结算公式：

```
击倒概率 = min(0.95, 主攻方口才 / max(1, 主攻方口才 + 守方口才) × (1 + 圣眷差加成))
反击概率 = min(0.5, 守方口才 / max(1, 主攻方口才 + 守方口才) × (守方圣眷 / (守方圣眷 + 10)) × 0.5)
未击倒概率 = max(0.05, 1 - 击倒概率 - 反击概率)
```

NPC不会主动消耗圣眷使用防御技能——圣眷留给日常资源运营。

## 十六、NPC家族间互动

### 16.1 NPC联姻互动

**触发条件：** 双方家族适龄未婚角色存在 且 关系≥30

**频率：** 每个NPC家族每年最多发起1次NPC间联姻

**选择逻辑：** 按7.2节目标家族评分公式选择

### 16.2 NPC结交互动

NPC家族之间每月有概率自动结交：

**触发概率：** 5%/月（每对家族独立判定）

**效果：** 关系+2~+5（消耗1点资源点 + 5贯金钱）

### 16.3 NPC商议互动

NPC家族之间竞争城市建筑槽位时，会互相发起商议：

**触发条件：** 入驻上限未满 且 目标建筑有空槽被其他NPC占用

**频率：** 每年最多1次

**结果：** 按商议成功率公式自动判定

## 十七、难度分级

AI行为逻辑可根据难度等级调整，同时NPC家族获得的各类资源有全局加成系数：

| 参数 | 简单 | 普通 | 困难 |
|------|------|------|------|
| NPC资源点利用率（每月实际消耗占可用资源点的比例） | 60% | 80% | 95% |
| NPC攻击频率 | 偏低(50%) | 正常(75%) | 积极(100%) |
| NPC防御投入 | 偏少 | 正常 | 充分 |
| NPC联姻/结交频率 | 低 | 正常 | 高 |
| NPC策略失误率 | 20% | 10% | 3% |
| NPC截留比例修正 | -5% | 标准 | +5% |
| **NPC全局资源加成** | ×0.8 | ×1.0 | ×1.2 |
| **NPC学识成长加成** | 80% | 100% | 120% |
| **NPC建筑产出加成** | ×0.8 | ×1.0 | ×1.2 |
| **NPC俸禄加成** | ×0.8 | ×1.0 | ×1.2 |
| **NPC声望获取加成** | ×0.8 | ×1.0 | ×1.2 |
| **NPC圣眷获取加成** | ×0.8 | ×1.0 | ×1.2 |

**全局资源加成说明：**

- NPC家族所有来源获得的金钱/声望/圣眷，结算时乘以对应难度系数
- 包括：建筑产出、俸禄、官职声望加成、党派分红、税收截留等一切来源
- 不影响：消耗类操作（攻击消耗不变、升级费用不变）
- 加成系数在每月结算阶段统一乘算，不改变公式本身

- 策略失误率：NPC做出非最优选择的概率（如攻击高稳固值目标而非低稳固值目标）
