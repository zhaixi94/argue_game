# 官场党争复仇游戏 — 场景与UI设计文档

> **引擎**：Godot 4.x
> **UI原则**：所有界面使用静态场景树（.tscn），禁止代码动态生成布局；节点位置/尺寸在编辑器中固定，运行时仅更新文本和数据

---

## 一、场景总览

游戏场景按功能分为5大类，共25个场景：

| 类别 | 场景数 | 说明 |
|------|--------|------|
| **全局场景** | 5 | 菜单、角色创建、主界面、结算、设置 |
| **家族场景** | 7 | 家族管理、角色、建筑、行动、小游戏 |
| **官场场景** | 5 | 官职、政绩、科举、中枢攻防 |
| **党派场景** | 4 | 党派总览、攻防、情报、私库 |
| **叙事场景** | 4 | 剧情、事件、任务、地图 |

## 二、全局场景

### 2.1 开始菜单

进入游戏的首个界面。

**布局：**

```
┌─────────────────────────────────────────────┐
│                                             │
│              [游戏标题/题字]                  │
│                                             │
│           ┌───────────────┐                 │
│           │   新游戏开始   │                 │
│           └───────────────┘                 │
│           ┌───────────────┐                 │
│           │   继续游戏     │  ← 有存档时显示  │
│           └───────────────┘                 │
│           ┌───────────────┐                 │
│           │   失败重开     │  ← 失败后        │
│           └───────────────┘                 │
│                                             │
│     [设置]              [制作组]             │
│                                             │
└─────────────────────────────────────────────┘
```

**功能要素：**

| 要素 | 设计 |
|------|------|
| **新游戏开始** | 进入角色创建界面 |
| **继续游戏** | 读取存档，回到主界面；无存档时此按钮灰置 |
| **失败重开** | 失败后可继承buff重新开局，进入重开设置界面 |
| **设置** | 音量、画质、自动存档频率等 |
| **背景** | 水墨风山水/朝堂剪影，随季节或时间缓慢变化 |

**失败重开设置界面：**

| 类型 | 显示内容 | 选择项 |
|------|---------|--------|
| **失败重开** | 当前获取倍率（声望获取×1.2, 金钱获取×1.2，叠加次数） | 确认开始 / 取消 |

### 2.3 主界面（回合主视图）

游戏最核心的常驻界面，玩家在此进行每回合的决策。

**布局：**

```
┌────────────────────────────────────────────────────────────┐
│ 顶部栏                                                      │
│ [年/季/月] [下一回合▶]  资源点:X/10  金钱:X贯  声望:X  圣眷:X  戒心:X │
├──────────┬─────────────────────────────────┬───────────────┤
│          │                                 │               │
│ 左侧     │         中央区域                 │    右侧       │
│ 快捷导航 │                                 │   信息面板    │
│          │  根据当前选中标签显示不同内容：    │               │
│ [家族]   │  - 家族建筑网格                   │  当月提醒     │
│ [官场]   │  - 官职/政绩槽                   │  待处理事件   │
│ [党派]   │  - 党派攻防面板                   │  任务进度     │
│ [地图]   │  - 地图总览                       │  关系变动     │
│ [人物]   │  - 角色卡详情                     │               │
│          │                                 │               │
├──────────┴─────────────────────────────────┴───────────────┤
│ 底部栏                                                      │
│ [剧情▶]  [任务]  [事件]  [存档]                              │
└────────────────────────────────────────────────────────────┘
```

**顶部栏：**

| 要素 | 说明 |
|------|------|
| **时间显示** | "景和三年·春·五月" 格式，点击展开详细时间轴（显示下次述职/科举/大朝会倒计时） |
| **下一回合** | 完成所有决策后点击推进；有未处理的必做操作时按钮灰置并提示 |
| **资源点** | 当前/上限，点击展开资源点分配详情 |
| **金钱** | 当前持有，文/贯自动换算 |
| **声望** | 当前值 + 衰减预警图标（当月衰减量>5时显示） |
| **圣眷** | 中枢后显示，点击展开衰减/戒心详情 |
| **戒心** | 中枢后显示，数值变色：0~29白/30~59黄/60~79橙/80~100红 |

**左侧导航：**

| 标签 | 切换中央区域显示 | 对应系统 |
|------|-----------------|---------|
| 家族 | 家族总览视图 | 家族系统 |
| 官场 | 官职与政绩视图 | 官场系统 |
| 党派 | 党派攻防视图 | 党争系统 |
| 地图 | 州县地图视图 | 地图系统+建筑系统 |
| 人物 | 角色卡列表/详情 | 人物/角色系统 |

**右侧信息面板：**

| 要素 | 说明 |
|------|------|
| **当月提醒** | 本月到期的任务、即将触发的述职/科举等 |
| **待处理事件** | 触发后未处理的事件数量，点击进入事件场景 |
| **任务进度** | 主线任务倒计时（红色预警），支线/党派任务简要 |
| **关系变动** | 本回合关系值变动摘要（联姻成功、结交结果等） |

**底部栏：**

| 要素 | 说明 |
|------|------|
| **剧情▶** | 有可播放剧情时高亮闪烁，点击进入剧情场景 |
| **任务** | 打开任务列表 |
| **事件** | 打开事件处理界面（有未处理事件时标数字） |
| **存档** | 手动存档/读档 |

**系统关联：**

```
主界面 ← 所有系统的数据汇总展示
  ├→ 资源系统：顶部栏资源实时显示
  ├→ 时间系统：下一回合推进 + 时间节点提醒
  ├→ 家族系统：左侧"家族"标签
  ├→ 官场系统：左侧"官场"标签
  ├→ 党争系统：左侧"党派"标签
  ├→ 地图系统：左侧"地图"标签
  ├→ 人物系统：左侧"人物"标签
  ├→ 事件系统：底部"事件"按钮 + 右侧待处理
  ├→ 剧情系统：底部"剧情"按钮
  └→ 任务系统：底部"任务"按钮 + 右侧任务进度
```

### 2.4 回合结算界面

每回合点击"下一回合"后弹出，展示本回合所有结算结果。

**布局：**

```
┌──────────────────────────────────────┐
│         景和三年·春·五月 回合结算      │
├──────────────────────────────────────┤
│                                      │
│  【资源变动】                         │
│   金钱    +35贯（建筑20+俸禄15）      │
│   声望   +12（建筑产出）  -6（衰减）  │
│   圣眷    +3（官职产出）  -1（衰减）  │
│   戒心    +0.5（圣眷衰减转化）         │
│                                      │
│  【能力成长】                         │
│   沈枢(私塾) 学识 +1.2 → 45.2        │
│   陆芷薇(书院) 学识 +0.8 → 38.5      │
│                                      │
│  【家族关系变动】                     │
│   与周家关系 +3（结交成功）            │
│   与苏家基础关系 -0.5（党派衰减）      │
│                                      │
│  【政绩槽/稳固值变动】                │
│   知县事 政绩槽 +6.8% → 54.2%        │
│                                      │
│  【事件】                             │
│   ▶ 故人来访（需处理）                │
│                                      │
│            [确认]                      │
└──────────────────────────────────────┘
```

**显示规则：**

| 板块 | 显示条件 | 内容 |
|------|---------|------|
| 资源变动 | 始终显示 | 各资源增减来源明细 |
| 能力成长 | 有角色成长时显示 | 角色名+建筑+成长值+当前值 |
| 家族关系变动 | 有变动时显示 | 关系增减及原因 |
| 政绩槽/稳固值变动 | 担任官职时显示 | 填充/攻防结果 |
| 事件 | 有事件触发时显示 | 事件名称，点击进入事件场景 |

- 无事件触发时，点击[确认]直接进入下一回合
- 有事件触发时，[确认]变为[处理事件]，进入事件场景

**特殊结算界面：**

| 类型 | 触发时机 | 额外内容 |
|------|---------|---------|
| **月度结算** | 每回合 | 上述标准内容 |
| **季度结算** | 每3月 | 增加：税收汇总、党派国库上缴对比、繁荣度变动 |
| **年度述职** | 每12月 | 替换月度结算，显示政绩槽评级结果（甲/乙/丙/丁）及后果 |
| **科举结算** | 科举年 | 三试各阶段独立结算界面 |

```
│                                             │
│  [确认]              [返回]                  │
└─────────────────────────────────────────────┘
```

**家族建筑 vs 城市建筑差异：**

| 维度 | 家族建筑 | 城市建筑 |
|------|---------|---------|
| 进入方式 | 家族总览→点击建筑 | 地图视图→点击州城→点击建筑 |
| 建造/升级 | 玩家操作，消耗金钱 | 不可建造，升级由繁荣度+知州决定 |
| 槽位竞争 | 无（家族专属） | 有（与NPC家族竞争，需商议） |
| 入驻消耗 | 无 | 金钱类无消耗；声望类/成长类每回合消耗金钱 |
| 额外信息 | — | 所属州城、繁荣度、司户参军加成 |

**系统关联：**

```
建筑详情
  ├→ 建筑系统：产出公式计算、升级
  ├→ 角色系统：角色放置/移出
  ├→ 资源系统：资源点投入、产出预览
  └→ 地图系统（城市建筑）：繁荣度、州城信息
```

| 游戏阶段 | 时间跨度 | 解锁场景 | 解锁系统/操作 |

| **第4期·地方晋阶** | 120月(16→26岁) | 官职总览、政绩槽详情、联姻/结交/商议、事件场景、任务界面、地图视图 | 官场系统(地方)、家族行动、党派加入 |

| **高亮引导** | 当前教学步骤对应的按钮/区域高亮闪烁 |
| **提示气泡** | 首次进入各界面时弹出简短说明 |
