# 官场党争复仇游戏 — 场景与UI设计文档

> **引擎**：Godot 4.x
> **UI原则**：所有界面使用静态场景树（.tscn），禁止代码动态生成布局；节点位置/尺寸在编辑器中固定，运行时仅更新文本和数据

---

## 一、场景总览

游戏场景按功能分为5大类，共26个场景：

| 类别 | 场景数 | 说明 |
|------|--------|------|
| **全局场景** | 5 | 菜单、角色创建、主界面、结算、系统 |
| **家族场景** | 6 | 家族管理、角色、建筑、行动、小游戏 |
| **官场场景** | 4 | 官职、政绩、科举、中枢攻防 |
| **党派场景** | 4 | 党派总览、攻防、情报、私库 |
| **叙事场景** | 5 | 剧情、事件、任务、地图、城市 |

```

│ 顶部栏                                                      │

│                                            │               │
│                 中央区域                     │  当月提醒     │
│                                            │  待处理事件(X)│
│              地图（默认视图）                │  任务进度     │
│                                            │  关系变动     │
│                                            │               │
│                                            │               │
│ [菜单]  [下一回合▶]                        │               │
└────────────────────────────────────────────┴───────────────┘
```

**菜单展开后：**

```
┌────────────────────────────────────────────────────────────┐
│ 顶部栏                                                      │
│ [年/季/月]  资源点:X/10  金钱:X贯  声望:X  圣眷:X  戒心:X │
├────────────────────────────────────────────┬───────────────┤
│  ┌─────────────┐                           │               │
│  │ [家族] [官场]│                           │  当月提醒     │
│  │ [党派] [地图]│         中央区域           │  待处理事件(X)│
│  │ [人物] [任务]│                           │  任务进度     │
│  │ [事件] [剧情]│                           │  关系变动     │
│  │ [系统]      │                           │               │
│  └─────────────┘                           │               │
│ [菜单]  [下一回合▶]                        │               │
└────────────────────────────────────────────┴───────────────┘
```

**顶部栏：**

| 要素 | 说明 |
|------|------|
| **时间显示** | "景和三年·春·五月" 格式，点击展开详细时间轴（显示下次述职/科举/大朝会倒计时） |
| **资源点** | 当前/上限，点击展开资源点分配详情 |
| **金钱** | 当前持有，文/贯自动换算 |
| **声望** | 当前值 + 衰减预警图标（当月衰减量>5时显示） |
| **圣眷** | 中枢后显示，点击展开衰减/戒心详情 |
| **戒心** | 中枢后显示，数值变色：0~29白/30~59黄/60~79橙/80~100红 |

**菜单按钮（左下角）：**

| 菜单项 | 说明 |
|--------|------|
| **下一回合** | 完成所有决策后点击推进；有未处理的必做操作时按钮灰置并提示 |
| **家族** | 进入家族总览视图 |
| **官场** | 进入官职与政绩视图 |
| **党派** | 进入党派攻防视图（加入党派后解锁） |
| **地图** | 中央区域切回地图视图（默认即地图） |
| **人物** | 进入角色卡列表/详情 |
| **任务** | 打开任务列表；有可领取/即将到期任务时标数字 |
| **事件** | 打开事件处理界面；有未处理事件时标数字 |
| **剧情** | 有可播放剧情时高亮闪烁，点击进入剧情场景 |
| **系统** | 进入系统界面(2.5)，包含存档/读档和设置 |

**右侧信息面板：**

| 要素 | 说明 |
|------|------|
| **当月提醒** | 本月到期的任务、即将触发的述职/科举等 |
| **待处理事件** | 触发后未处理的事件数量，点击进入事件场景 |
| **任务进度** | 主线任务倒计时（红色预警），支线/党派任务简要 |
| **关系变动** | 本回合关系值变动摘要（联姻成功、结交结果等） |

**中央区域：**

默认显示地图视图，展示全国州县分布。点击菜单项后切换对应视图（地图/家族/官场/党派等）。

**系统关联：**

```
主界面 ← 所有系统的数据汇总展示
  ├→ 资源系统：顶部栏资源实时显示
  ├→ 时间系统：下一回合推进 + 时间节点提醒
  ├→ 家族系统：菜单"家族"
  ├→ 官场系统：菜单"官场"
  ├→ 党争系统：菜单"党派"
  ├→ 地图系统：菜单"地图"
  ├→ 人物系统：菜单"人物"
  ├→ 事件系统：菜单"事件"
  ├→ 剧情系统：菜单"剧情"
  └→ 任务系统：菜单"任务"
```

**功能要素：**

| 要素 | 说明 |
|------|------|
| **存档/读档页签** | 默认页签，10个存档位，每个显示存档时间、家族名、声望、当前官职 |

| **返回** | 关闭系统界面，回到主界面 |

## 八、游戏阶段与场景解锁

| 游戏阶段 | 时间跨度 | 解锁场景 | 解锁系统/操作 |
|---------|---------|---------|-------------|
| **第1期·新手教学** | 12月(13岁) | 主界面(简化)、家族总览(简化)、建筑详情、角色详情、学识小游戏 | 回合推进、建筑放置、资源点分配 |

| **第4期·地方晋阶** | 120月(16→26岁) | 官职总览、政绩槽详情、联姻/结交/商议、事件场景、任务界面、地图视图 | 官场系统(地方)、家族行动、党派加入 |
