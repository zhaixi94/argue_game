# 官场党争复仇游戏 — 党派与党争系统设计文档

---

## 一、党派基础设定

- 2个对立党派：**畿党**与**淮党**（固定，不可完全消灭，只会式微）
- 主控家族加入党派后开启党派系统
- 无党派家族（顾、梁、何）可被拉拢加入任一党派

### 两党根基差异

| 维度 | 畿党 | 淮党 |
|------|------|------|
| **根基所在** | 京畿路 | 江南路 |
| **核心优势** | 靠近中枢，政治资源强 | 经济富庶，金钱资源强 |
| **核心家族** | 陆家(开封)、周家(河南) | 苏家(江宁)、钱家(苏州) |
| **楔子家族** | 林家(江宁)、赵家(广州) | 方家(应天)、陈家(广州) |
| **党派领袖风格** | 政治型提携，替人整朝服、解围发言 | 经济型提携，送家具仆从、批档库条子 |

### NPC家族党派分布

| 党派 | 京畿路 | 江南路 | 岭南路 | 合计 |
|------|--------|--------|--------|------|
| **畿党** | 2（陆、周） | 1（林） | 1（赵） | 4 |
| **淮党** | 1（方） | 2（苏、钱） | 1（陈） | 4 |
| **无党派** | 0 | 1（顾） | 2（梁、何） | 3 |

- 主角家族加入畿党或淮党后改变4v4平衡

## 二、党派领袖

- 一个党派有多位领袖
- 前任领袖死亡/贬官/罢免后，从以下三个维度判定接任：
  - 官职高低
  - 声望
  - 党派贡献

### 2.1 领袖接任判定公式

当前领袖死亡/贬官/罢免时，该党派所有正三品以上（含）在任中枢官员参与接任判定，综合评分最高者成为新领袖：

```
接任评分 = 官职维度分 + 声望维度分 + 贡献维度分
```

**官职维度分：**

```
官职维度分 = 品阶权重 × 100

品阶权重：
  正一品 = 10
  从一品 = 9
  正二品 = 8
  从二品 = 7
  正三品 = 6（正三品为参与门槛，权重最低）
```

- 散官不参与接任判定，必须在任官员
- 官职维度最高1000分（正一品×100）

**声望维度分：**

```
声望维度分 = 家族声望 × 0.05（上限200分）
```

- 声望维度最高200分，避免声望碾压官职维度
- 家族声望≥4000即可拿到满分

**贡献维度分：**

```
贡献维度分 = 党派贡献 × 0.5（上限300分）
```

- 贡献维度最高300分，长期经营党派的回报
- 党派贡献≥600即可拿到满分

**评分示例：**

| 候选人 | 官职 | 声望 | 党派贡献 | 评分 |
|--------|------|------|----------|------|
| A | 正二品(800) | 5000(200) | 200(100) | 1100 |
| B | 从二品(700) | 3000(150) | 600(300) | 1150 |
| C | 从二品(700) | 6000(200) | 100(50) | 950 |

- B虽然官职低于A，但贡献远高于A，总分反超
- 官职权重最大（最高1000），但声望+贡献合计可达500，非官职最高者仍有机会

**同分处理：**

1. 官职维度分高者优先
2. 仍相同则党派贡献高者优先
3. 仍相同则随机决定

## 三、党派资源

| 资源 | 来源 | 用途 |
|------|------|------|
| **党派声望** | 中枢官员官职产出；成员家族上缴声望；地方官员政绩影响；被罢免/贬官则降低；每月自然衰减1%(最低月衰减1点，下限100) | 解锁党派机构；兑换党派资源点 |

**党派声望产出量化：**

| 来源 | 产出规则 |
|------|---------|
| **中枢官员官职产出** | 每月：正一品+30, 从一品+20, 正二品+15, 从二品+10, 正三品+5 |
| **成员家族上缴声望** | 每月：成员家族可自愿上缴声望，1点家族声望=1点党派声望（NPC按AI逻辑自动上缴） |
| **地方官员政绩影响** | 每年述职时：甲等+10, 乙等+0, 丙等-10, 丁等-20 |
| **罢免/贬官** | 每次罢免-50, 每次贬官-20 |
| **党派资源点** | 消耗党派声望兑换；党派机构产出 | 攻击/防守官职稳固值；补充己方地方官政绩槽；投入党派机构提升产出；舆论机构buff |

### 党派资源点兑换规则

```
兑换比例：50党派声望 → 1党派资源点
每月兑换上限：党派声望 / 100（向下取整，最少1点）
```

- 兑换消耗的党派声望不可退还
- 党派资源点不跨月累积，每月初清零（与家族资源点同理）
- 党派机构每回合基础产出1党派资源点（需解锁后）

## 四、党派机构（党派声望达标后解锁）

| 机构 | 解锁条件 | 机制 | 产出 |
|------|----------|------|------|
| **谍报机构** | 党派声望≥200 | 抽卡制，每回合随机产出情报卡，投入党派资源点增加产出 | 多类型情报卡，配合资源点攻击非己方官员 |
| **舆论机构** | 党派声望≥500 | Buff制，投入党派资源点产生buff，持续3回合 | 己方增益buff / 敌方减益buff，投入越多越强 |
| **私库** | 默认可用 | 金钱库，己方官员收入按比例进入；可主动存入换党派贡献 | 上缴皇帝→党派领袖获圣眷(10贯=1圣眷)；每年按官位+贡献分红给成员→提高忠诚 |

### 4.1 谍报机构产出规则

- 每回合基础产出(1 + level)张情报卡（类型随机），等级越高产出越多
- 投入党派资源点增加产出：

| 投入党派资源点 | 额外产出 |
|---------------|---------|
| 1点 | +1张 |
| 2点 | +2张 |
| 3点 | +3张 |

- 每回合最多投入3点党派资源点
- 每回合情报卡产出上限 = 1 + level + 3（即1级上限5张，2级上限6张，3级上限7张，4级上限8张，5级上限9张）
- 情报类型随机：检举情报40% / 弹劾情报25% / 离间情报35%
- 情报等级按概率：初级60% / 中级30% / 高级10%
- 截留情报不在此产出，由税收截留机制触发
- 情报卡持有上限：由家族暗阁等级决定（详见建筑系统设计文档→情报类建筑）

### 4.2 机构升级规则

- 每月最多升级1个机构，每次升1级
- 升级消耗：私库金钱 + 党派声望 + 党派资源点
- 最高5级

| 机构 | 解锁条件 | 升级消耗(每级) | 每级效果提升 |
|------|---------|---------------|-------------|
| **谍报机构** | 党派声望≥200 | 100贯私库金钱 + 50党派声望 + 3党派资源点 | 基础产出+1张情报卡/月 |
| **舆论机构** | 党派声望≥500 | 100贯私库金钱 + 50党派声望 + 3党派资源点 | buff强度+5%/级，持续时间+1回合/级 |
| **私库** | 默认可用 | 100贯私库金钱 + 50党派声望 + 3党派资源点 | 抽成上限+2%/级(10%→12%→14%...) |

### 4.3 私库抽成规则

- 党派领袖可设置私库抽成比例（0%~10%）
- 抽成来源：该党派所有官员的每月总收入（税收截留+俸禄）
- 抽成比例越高，该党派中NPC家族的党派忠诚度下降越快
- 具体数值见建筑系统设计文档 → 税收系统 → 党派私库抽成

### 4.4 舆论机构buff效果

| buff类型 | 效果 | 持续 | 投入 |
|----------|------|------|------|
| **清名**（己方增益） | 己方所有中枢官员稳固值防御减免+10%(+5%/舆论等级) | 3回合(+1/舆论等级) | 2党派资源点 |
| **流言**（敌方减益） | 敌方所有中枢官员攻讦成功率+10%(+5%/舆论等级) | 3回合(+1/舆论等级) | 3党派资源点 |
| **颂德**（己方增益） | 己方所有家族圣眷获取+10%(+5%/舆论等级) | 3回合(+1/舆论等级) | 3党派资源点 |
| **疑云**（敌方减益） | 敌方所有家族戒心每月额外+2(+1/舆论等级) | 3回合(+1/舆论等级) | 4党派资源点 |
| **风骨**（己方增益） | 己方大朝会连线实体血量+50(+20/舆论等级) | 3回合(+1/舆论等级) | 5党派资源点 |
| **倾轧**（敌方减益） | 敌方党派忠诚度每月额外-3(-1/舆论等级) | 3回合(+1/舆论等级) | 5党派资源点 |

- 同类型buff不可叠加，重复投入只刷新持续时间
- 己方增益和敌方减益可同时生效

## 五、党派行动

| 行动 | 消耗 | 效果 |
|------|------|------|
| 攻击敌方官职 | 攻讦/检举/离间/弹劾 | 攻击非己方官职稳固值，获得党派贡献 |
| 帮助非主控家族 | 家族资源点 | 获得党派贡献 + 提高该家族党派忠诚度 |
| 党派领袖攻防 | 党派资源点 | 帮助非主控家族防御获党派忠诚度 |
| 拉拢策反 | 家族资源点/金钱 | 降低目标家族与所属党派忠诚度 |

## 六、党派贡献

### 6.1 贡献获取

| 来源 | 获取量 |
|------|--------|
| 党派任务完成 | 任务基础奖励(3~8点，按任务难度) |
| 党争行动成功 | 攻讦/检举成功+3，离间成功+4，弹劾成功+5 |
| 上缴金钱 | 每上缴10贯→+1贡献 |
| 上缴声望 | 每上缴10点声望→+1贡献 |

- 党争行动失败不获得贡献
- 上缴金钱/声望为主动行为，非自动扣除
- 党派任务参见主文档任务系统→党派任务模板

### 6.2 贡献消耗

| 用途 | 说明 |
|------|------|
| 换取情报 | 在谍报机构获取情报卡，消耗党派贡献领取 |
| 兑换资源 | 换算为金钱/声望（从私库/党派声望扣除） |
| 年度分红 | 每年初从私库分配，见6.3节 |

### 6.3 年度分红

- 每年初，私库总额的30%用于分红，剩余70%保留
- 各成员家族分红金额 = 分红总额 × 该家族分配权重 / 全体分配权重之和
- 分配权重 = 官位权重 + 贡献权重

| 官职品级 | 官位权重 | 贡献权重 |
|----------|----------|----------|
| 正一品 | 5 | 贡献值/20(向上取整) |
| 从一品 | 4 | 贡献值/20(向上取整) |
| 正二品 | 3 | 贡献值/20(向上取整) |
| 从二品 | 2 | 贡献值/20(向上取整) |
| 正三品 | 1 | 贡献值/20(向上取整) |
| 无中枢官职 | 0 | 贡献值/20(向上取整) |

- 获得分红的家族党派忠诚度+3

### 情报卡领取消耗

| 情报等级 | 党派贡献消耗 |
|----------|-------------|
| 初级(1级) | 5点 |
| 中级(2级) | 15点 |
| 高级(3级) | 30点 |

- 谍报机构每回合产出的情报卡进入党派情报池，所有成员可见
- 家族需消耗党派贡献从情报池中领取情报卡到自己家族
- 领取后该情报卡从党派情报池中移除，其他家族不可再领取
- 不领取的情报卡在月末清零（不跨月保留）
- 截留情报无需消耗党派贡献，触发时直接获得

## 七、党派忠诚

| 要素 | 设计 |
|------|------|
| **初始值** | NPC家族加入时默认70（0-100） |
| **自然衰减** | 每月-1（基础衰减，无法避免） |
| **提升** | 帮助该家族防御/分红等行为 |
| **降低** | 被拉拢/利益受损/私库抽成等 |
| **策反阈值** | 忠诚度<30时可被拉拢策反 |
| **叛离** | 忠诚度=0自动退出并加入对立党派 |

## 八、稳固值攻防系统

### 8.1 稳固值基本规则

| 要素 | 设计 |
|------|------|
| **稳固值** | 每个中枢官职拥有（与官职绑定而非角色），范围无硬下限，初始值60 |
| **提升己方** | 未被攻击的回合可消耗圣眷或声望提升：3圣眷兑换1稳固值，或50声望兑换1稳固值；每回合最多提升10点 |
| **攻击敌方** | 消耗资源攻击非己方官职，降低其稳固值。共4种攻击方式：攻讦、检举、离间、弹劾 |
| **防御机制** | 月初预设防御姿态，消耗资源点+圣眷降低当月受到的伤害 |
| **罢免/贬官** | 稳固值降至0及以下时：从二品以上触发大朝会辩论；正三品直接贬官（降一阶）或罢免（由戒心等级决定，戒心≥80则罢免，否则贬官） |
| **空缺升迁** | 出现空缺官职时，可消耗圣眷+声望请求升迁调任 |

### 8.2 地方官政绩槽攻防规则

- 地方官政绩槽只能被攻讦和检举攻击，无法被离间或弹劾（离间/弹劾仅对中枢官员生效）

| 攻击方式 | 消耗 | 效果 | 成功率 |
|----------|------|------|--------|
| **攻讦** | 1~3资源点 | 降低目标政绩槽5+(投入-1)×3个百分点 | 70%+口才/3 |
| **检举** | 1张检举情报+1~3资源点 | 降低目标政绩槽5+情报等级×5+(投入-1)×3个百分点 | 100%（需情报） |

### 8.3 中枢官职攻击方式总览

| 攻击方式 | 消耗 | 成功率 | 伤害/效果 | 额外效果 | 每回合限制 |
|----------|------|--------|-----------|----------|-----------|
| **攻讦** | 1~3资源点 | 70%+口才/3（上限100%） | 5 + (投入资源点-1)×3 | 无 | 每目标1次 |
| **检举** | 1张检举情报+1~3资源点 | 100%（需情报） | 5 + 情报等级×5 + (投入资源点-1)×3 | 无 | 每目标1次 |
| **离间** | 1张离间情报+1~3资源点 | 比较双方圣眷（见公式） | 稳固值伤害5 + (投入资源点-1)×2 | 成功时：目标圣眷-5×情报等级，目标戒心+3×情报等级 | 每目标1次 |
| **弹劾** | 3~5资源点+10圣眷+1张弹劾情报 | 3%+圣眷/10+情报等级×5（上限60%） | 60（成功时） | 成功时额外：目标戒心+10 | 每目标1次 |

### 8.4 各攻击方式详细规则

**攻讦：高频低伤**

```
成功率 = min(100%, 70% + 发起方口才 / 3)
伤害 = 5 + (投入资源点 - 1) × 3     （投入1~3点）
```

- 成功率受发起方口才影响，口才90时达到100%
- 攻讦失败则无伤害，消耗的资源点不退还
- 不需要任何情报卡，是最基础的攻击手段

**检举：必中中伤**

```
成功率 = 100%（必须消耗1张检举情报）
伤害 = 5 + 情报等级 × 5 + (投入资源点 - 1) × 3     （投入1~3点）
```

- 检举情报等级：初级1/中级2/高级3，对应基础伤害10/15/20
- 无情报则无法发起检举
- 伤害稳定可预期，是中期主要输出手段

**离间：圣眷打击**

```
成功率 = min(90%, 发起方家族圣眷 / (发起方家族圣眷 + 目标官员所在家族圣眷) × 100%)
稳固值伤害 = 5 + (投入资源点 - 1) × 2     （投入1~3点）
```

- 成功时额外效果：目标官员所在家族圣眷 -5×情报等级，目标戒心 +3×情报等级
- 离间的核心价值不在稳固值伤害，而在削圣眷、涨戒心——为弹劾或罢免铺路
- 离间失败则无任何效果，消耗的资源点和情报不退还

**弹劾：孤注一掷**

```
成功率 = min(60%, 3% + 发起方圣眷 / 10 + 弹劾情报等级 × 5)
伤害 = 60（成功时）
```

- 成功时额外效果：目标戒心+10
- 弹劾失败则无伤害，消耗的资源点、圣眷、情报不退还
- 弹劾成功率上限60%，即使圣眷极高也无法保证成功
- 可通过离间先削减对方圣眷、再弹劾提升成功率——离间+弹劾是核心combo
- 弹劾情报等级加成：初级+5%/中级+10%/高级+15%

### 8.5 弹劾与大朝会的关系

弹劾≠大朝会。弹劾仅降低稳固值，若稳固值降至0则按品级触发后续流程：
- 从二品以上：触发大朝会辩论（大朝会完整规则见主文档→大朝会机制）
- 正三品：直接贬官/罢免（视戒心等级）

### 8.6 防御机制：月初预设

每月初，玩家可为己方每个中枢官职预设防御姿态，消耗资源点+圣眷降低当月受到的所有攻击伤害：

```
伤害减免 = 资源点减免 + 圣眷减免
资源点减免 = 投入资源点 × 10%（最多投入3点，减免30%）
圣眷减免 = 投入圣眷 / 5 × 10%（最多投入30点，减免60%）
总减免上限 = 80%
```

| 投入 | 减免 |
|------|------|
| 1资源点 | 10% |
| 2资源点 | 20% |
| 3资源点 | 30% |
| 5圣眷 | 10% |
| 10圣眷 | 20% |
| 15圣眷 | 30% |
| 20圣眷 | 40% |
| 25圣眷 | 50% |
| 30圣眷 | 60% |

- 资源点减免和圣眷减免可叠加，但总减免不超过80%
- 防御消耗在月初一次性扣除，对当月所有攻击生效
- 未设防御的官职受到攻击时按原始伤害计算
- NPC官员同样会根据自身资源设置防御

### 8.7 受伤系数修正

戒心等级影响稳固值被攻击时的受伤系数：

| 戒心区间 | 受伤系数 |
|----------|---------|
| 0~59 | ×1.0 |
| 60~79 | ×1.2 |
| 80~100 | ×2.0 |

```
实际伤害 = 计算伤害 × (1 - 防御减免) × 受伤系数
```

## 十、贬官机制

- 大朝会贬官：降一阶（如从二品→正二品），保留官职
- 正三品稳固值归零贬官：降一阶，保留官职
- 戒心触发贬官：按戒心降低规则执行（每贬一阶降5戒心，单次最多降30）
- 罢免：削职为民，失去品阶和官职
- 贬官后可复出，但需要重新积累

## 十一、中枢玩法循环

```
月初：预设防御姿态（资源点+圣眷→减伤）
              ↓
回合中：选择攻击方式
  ├─ 攻讦：高频低伤，磨血消耗
  ├─ 检举：必中中伤，需情报
  ├─ 离间：削圣眷涨戒心，为弹劾铺路
  └─ 弹劾：孤注一掷，低成功率高伤害
              ↓
敌方稳固值=0 → 从二品以上：触发大朝会辩论
             → 正三品：直接贬官/罢免（视戒心等级）
              ↓
贬官/罢免 → 产生空缺
              ↓
消耗圣眷+声望 → 请求升迁填补空缺
```

## 十二、党派覆灭与平衡规则

- 固定2个党派不可完全消灭，只会式微
- 党派声望<500时进入式微状态，该党派所有机构停产，成员忠诚度每月额外-2；声望恢复至500以上时自动退出式微状态
- 但只要有一名从三品以上官员在任，党派就不会消亡
- **全员退出中枢**：当某党派所有成员均不在中枢任职时
  - 若该党派为主角所在党派 → 触发失败结局
  - 若该党派非主角所在党派 → 触发瓦解复仇结局
- **主角加入前平衡机制**：主角加入党派之前，声望较低的党派每回合额外获得资源补充，确保两个党派不会因AI行为失衡导致一方崩盘
  - 补充公式：`每回合额外获得 = max(0, (对方党派声望 - 本党派声望) × 0.1)` （党派声望直接增加此数值；同时该党派所有成员家族各获得等量金钱，单位：贯）
  - 例：甲党声望800，乙党声望500 → 乙党每回合额外获得30点党派声望，乙党所有成员家族各获得30贯金钱
  - 声望差≤50时不触发（双方基本平衡）
  - 主角加入任一党派后此机制立即停止

### 14.6 buff叠加规则

- 所有百分比buff统一采用乘法叠加：公式计算完成后 × (1 + buff数值累加)
  - 如：声望获取+10%(太师) + +15%(丞相) → 基础值 × (1 + 0.10 + 0.15) = 基础值 × 1.25
- 同类百分比buff上限：单个维度总加成不超过50%，超过部分截断
- 减益(戒心debuff)在所有加成之后计算：最终值 = 基础值 × (1 + 加成合计) × (1 - 减益合计)
- "己方党派所有家族"类buff与同家族buff共享50%上限
- 固定值buff（如大朝会血量+50、戒心每月-2）不参与百分比上限计算
- 同类型事件buff不叠加，取最高值（如两个"声望获取+10%"事件buff同时存在，只生效+10%）
- 不同类型buff可共存（如声望获取+10%与金钱产出+15%互不影响）
- **稳固值伤害计算顺序**：
  1. 先计算基础伤害
  2. 乘以(1 - 主动防御减免)，主动防御上限80%
  3. 再乘以(1 - 官职buff防御减免)，官职buff防御属于独立层，不占用主动防御80%上限
  4. 再乘以受伤系数
  5. **等效总减免上限99%**：若上述多层减免叠加后等效减免超过99%，则将最终伤害设为不低于基础伤害的1%（即无论防御多高，每次攻击至少造成1%基础伤害）
